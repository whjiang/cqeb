　　码表来自作者本人，即付东升老师。此时是最新版本。
　　付东升老师毕业于辽宁中医药大学，是超强系列两笔输入法的作者，付老师坚持只要署名权，不求利。十数年来极少数的还在坚持三十键两笔的人之一。

　　首先：超强系列用v键来引导部分符号的输入，所以不需要点开符号面板，比如，一个熟练的人的手里，问号键一定是按vv，两键上问号符。
　　其次：本皮肤的高度是可调节的，但是考虑到大多数情况，本主题提供了几个高度版本，最低的是默认版本，主高度为50，你可以自己找到自己喜欢的高度后也可以删了其它的。
　　至于方案，选定一个之后可以全部删了其它的，如以cqlb开头的文件或带有cqlb的都是超强两笔的文件。
　　如果删出的东西想恢复，可以去设置里面恢复出厂设置。


　　【功能（重要）】：
　　1、i键：引导全拼或者简拼，并进行反查。超强系列遇到一个字不会读可以用先打i键代替读音，后加形打出来，或者超强两笔的用i键引导一个全拼打出这个字。
　　2、键盘上可见的符号和数字可长按也可上滑得到。
　　3、A-M键位上面左右滑动、可以使光标左右移动一个位置。
　　4、空格键上滑是一个强制缩进，目前同文平台在内测，未稳定。
　　5、空格键左右滑是光标上下移动。
　　6、shift键左滑是清行。
　　7、X长按或上滑是剪切。C长按或上滑是复制，V长按或上滑是粘贴。A是全选，D上滑是日期下滑是时间。
　　8、删除键上滑关闭开启候选栏，左滑打开菜单。不喜欢五码，可以关闭注解功能，不会做四码版本。
　　9、※※※各个方案的助记皮肤调用：在自带的任何主题的情况下，点击候选栏空白处。
　　10、不喜欢符号干扰，可以关闭提示。
